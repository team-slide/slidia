What is Slidia?

📋 Manifest System

Slidia provides a comprehensive XML manifest system that catalogs all available firmware packages, themes, and tools for Y1 devices. This centralized approach makes it easy to discover and manage resources.

🔗 Repository Integration

Seamlessly integrates with GitHub repositories to provide real-time access to the latest firmware releases, updates, and community contributions.

📱 Device Compatibility

Built specifically for Y1 devices with comprehensive device type and model support, ensuring compatibility across all Y1 variants.

⚡ Performance Optimized

High-performance package discovery and management with intelligent caching and rate limiting for optimal GitHub API usage.

🌍 Cross-Platform

Platform-agnostic design that works seamlessly across Windows, macOS, and Linux systems.

🛠️ Developer Friendly

Clean APIs and well-documented structure make it easy for developers to extend and integrate with their own tools and applications.

🚀 Get Innioasis Updater

Download the desktop app for your system

🪟 Windows Installation

Download the Windows installer and follow the setup instructions. After installation, you'll need to install the MTK drivers for proper device communication.

🍎 macOS Installation

Follow the manual setup instructions in the new tab. After completing the setup, you can run the app with: python3 updater.py

An easier installation method for macOS is coming soon!

🐧 Linux Installation

Ubuntu is recommended for the best experience, but other distributions should work as well. Follow the instructions in the new tab to set up the required dependencies.

🪟 Windows ARM64

For Windows ARM64 users, more advanced users may be able to run the Linux version using Python, Libusb, android-platform-tools, USBIPD-win and WSLg. Ubuntu is recommended.

🖥️ ChromeOS/FydeOS

ChromeOS and FydeOS users may be able to run Innioasis Updater in Linux mode, but we have not tested this functionality. Ubuntu is recommended for the best experience.
